# Pikine Optique Vision

Site e-commerce moderne pour Pikine Optique Vision, spécialisé dans la vente de lunettes photochromiques, anti lumière bleue, anti reflet et lunettes de soleil.

## Structure du projet

- `index.html` - Page d'accueil responsive
- `administration/index.html` - Espace admin pour gérer produits et commandes
- `paiement/index.html` - Page de paiement et options Wave / à la livraison
- `css/style.css` - Styles globaux du site
- `js/config.js` - Configuration du numéro WhatsApp et nom du vendeur
- `js/data.js` - Produits par défaut, avis clients et stockage local
- `js/main.js` - Logique d'affichage public
- `js/admin.js` - Gestion des produits et commandes dans l'administration
- `assets/` - Dossier pour les images et les ressources visuelles

## Fonctionnalités

- Design moderne, premium, mobile first
- Bouton WhatsApp flottant sur toutes les pages
- Produits dynamiques avec description, prix, disponibilité et bouton WhatsApp
- Page de paiement avec options Wave et paiement à la livraison- Page produit détaillée avec galerie, caractéristiques et CTA premium
- Barre de filtres produits par catégorie et recherche- Administration locale avec ajout, modification, suppression de produits et visualisation des commandes
- Stockage local pour préserver les produits et commandes dans le navigateur

## Installation

1. Copier le projet dans votre serveur web local (`xampp`, `tomcat`, etc.)
2. Ouvrir `index.html` dans un navigateur moderne
3. Utiliser `administration/index.html` pour gérer les produits

## Personnalisation

- Mettre à jour `js/config.js` avec le vrai numéro WhatsApp du vendeur
- Ajouter de véritables images produits dans `assets/products/`
- Modifier les textes et les offres dans `index.html` et `js/data.js`

## Déploiement

Ce projet est prêt à être déployé sur GitHub Pages ou un hébergement statique standard.
